# Syntecxhub_Movie_Recommendation_System

Built a Movie Recommendation System using Machine Learning with content-based filtering. Performed data cleaning, feature engineering, and used cosine similarity to suggest similar movies based on genres, cast, and keywords. Implemented using Python, Pandas, and Scikit-learn.


## 📌 Overview
This project is a content-based movie recommender system built using Machine Learning.

## 📊 Features
- Data cleaning and preprocessing
- Feature engineering using text data
- Cosine similarity for recommendations

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn

## 🎯 How It Works
The system recommends movies based on similarity in:
- Genres
- Cast
- Keywords
- Overview

## 📸 Sample Output
Input: Avatar  
Output:
- Guardians of the Galaxy
- Star Trek
- The Avengers
